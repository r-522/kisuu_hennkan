\# 基数変換クイズ



\## 概要



\- 基数変換を四択で練習できる静的 Web アプリ。

\- 回答形式は `ア・イ・ウ・エ`。

\- 回答後に考え方と別解を表示。

\- エンドレスに問題を解ける。

\- DB 不要。

\- Vercel に静的サイトとしてデプロイ可能。



\## 主な機能



\- 2進数、8進数、10進数、16進数の相互変換。

\- ランダム出題。

\- 四択回答。

\- 正誤判定。

\- 解説表示。

\- 別解表示。

\- 正解数、回答数、正答率、連続正解数の表示。

\- スコアリセット。

\- レスポンシブ対応。

\## Vercel デプロイ概要



\- GitHub などにリポジトリを作成する。

\- Vercel にリポジトリをインポートする。

\- Framework Preset は `Other` を選択する。

\- Build Command は空欄にする。

\- Output Directory は `.` を指定する。

\- デプロイ後、発行された URL にアクセスする。



\## 実装方針



\- 内部値は10進整数で扱う。

\- 表示時に指定基数へ変換する。

\- 選択肢は正解1つと誤答3つで構成する。

\- 誤答は典型的な計算ミスを想定して生成する。

\- 解説は10進数経由の基本手順を中心にする。

\- 2進数、8進数、16進数の相互変換ではグルーピングによる別解を表示する。
